About Valkyr-IE

Valkyr-Information Extraction (Valkyr-IE) is a tool for Named Entity Recognition tasks. The tool uses:

• Language models trained with SoA datasets

• Language models trained with use-case datasets

• A wrapper for the General Architecture for Text Engineering (GATE) tool

API

Test our standard models through Valkyr-ie Swagger API .

Test our bio models through Valkyr-ie BIO Swagger API .

Test our GATE wrapper through Valkyr-ie GATE API .
